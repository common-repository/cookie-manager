== Description ==

[Cookie Manager](https://cookiemanager.deviware.com/) allows you to elegantly inform users that your site uses cookies and, in the PRO version, they can even configure the types of Cookies they consent to, helping you comply with the EU GDPR cookie law regulations.
You can also customize plugin texts and colors as well as download the user consents (PRO version).

For more information, check out plugin page at [Deviware](https://cookiemanager.deviware.com/).

= Features include: =

* Customizable texts
* Consent Log (PRO Feature)
* Link to Privacy Policy page (PRO Feature)
* Set the text and bar background colors
* .pot file for translations included (EN, ES)

== Installation ==

1. Install WP Cookie Manager either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Cookie Notice settings and set your options.
